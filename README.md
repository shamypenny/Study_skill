# Study_skill

## 定义：思维引导式学习助手skill

### 背景

#### 教育类AI产品市场严重饱和，同质化竞争激烈，而现有产品存在"直给答案代替思考"的核心缺陷——用户获取信息效率高，但知识沉淀率低、主动思维能力弱。本项目跳出独立产品赛道，以可插拔Skill形态切入，将苏格拉底式引导学习方法论封装为通用 AI Skill，可无缝接入任意 LLM 平台或教育产品，面向 K12 至全体学习用户，实现"一次设计、多场景复用"的平台化价值

### 核心方法论设计

#### 提炼"把大脑类比为LLM"的认知框架 —— 给条件（上下文）、给输入（问题）、产生输出（理解）、有反馈（检验），将主动学习理论与 AI 交互机制相结合；设计 4 阶段递进式引导流程：Define（明确问题）→ Understand & Decompose（分解任务）→ Practice（实践挑战）→ Reflect &Loop（反思循环），每阶段强制等待用户响应、逐步推进，从结构上杜绝"直接给答案"的信息轰炸模式

### Prompt 精细化设计

#### 为4个阶段分别设计差异化 Prompt策略 —— 第一阶段以"学什么/为什么学/学了能做什么"三问锚定学习目标并提炼学习问题陈述；第二阶段运用苏格拉底式提问（Socratic Questioning）引导用户主动思考，给出学习路线图而非直接内容；第三阶段设计"有效挑战"实践任务（难度略高于当前理解以形成认知拉伸）并提供方案评估反馈；第四阶段引导知识图谱更新与盲区识别，触发下一轮学习循环；全程以"脚手架提示"代替直接揭示答案

### Skill 架构设计

#### 将方法论封装为符合 Claude Code Skill 标准的插件格式，设计为零依赖、无状态绑定的轻量化结构，任何接入方（教育平台/个人 LLM用户/课程开发者）均可一键引入，无需额外开发成本；Skill 的交互设计遵循"每次只推进一个阶段"原则，保证在不同宿主平台下行为一致性
